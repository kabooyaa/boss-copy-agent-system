# BOSS Copy — High Level HTML Blocks

These are the standalone HTML code blocks used in the BOSS Copy High Level funnel pages. Each file is a self-contained HTML element to be pasted into an HL **Custom HTML** element on the relevant funnel page.

---

## Sales Page (2 blocks)

| File | Page | Placement |
|------|------|-----------|
| `boss_copy_sales_page_PART1.html` | Sales page | Everything **above** the video element — hero, problem, solution, how it works, output formats, plan cards, Platinum image showcase |
| `boss_copy_sales_page_PART2.html` | Sales page | Everything **below** the video element — 10 Reasons bullets, FAQ, final CTA, footer |

> The video itself is a native HL Video element placed between PART 1 and PART 2.
